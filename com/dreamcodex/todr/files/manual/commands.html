COMMAND KEY REFERENCE
Function Keys
F1Load Game
F2Save Game
F5Switch between Fullscreen and Floating window mode
F12Quit ToDR
Menu Keys
ESCExit Menu
1New Game
2Load Game
3Load Party & Generate New Dungeon
4Save Game
XGame Options
ENTERConfirm Menu Selection (where needed)
BKSPCErase One Text Letter (when naming character or game file)
DELErase All Text (when naming character or game file)
Common In-Game Keys
/Help
1Player Info
  • Use LEFT/RIGHT arrows to change displayed player
2Party Info
3Monster Info (in combat only)
4Quest Info
OChange Party Order
MView Map
UUse Item
  • Select item to use with UP/DOWN arrows
  • Use selected item with U key
  • Cancel with ESC key
IView Inventory
  • Use LEFT/RIGHT arrows to change displayed player
WSwap Equipt Weapon (when viewing Player Info)
XDestroy Item/Equipment
  • Use LEFT/RIGHT arrows to select player who will destroy item
  • Use UP/DOWN arrows to select item to destroy
  • Press X key to destroy item
  • Press Y key to confirm destruction, or N to cancel
  • Press ESC to exit destroy screen
ESCCancel Current Action/Show Main Menu
Dungeon View Keys
UPMove party forward
DOWNTurn party around
LEFTTurn party left
RIGHTTurn party right
BBreak Through Door (moves whole party into room)
LListen At Door
SSearch Hallway For Hidden Doors
Room View Keys
UPMove party out North exit
DOWNMove party out South exit
LEFTMove party out West exit
RIGHTMove party out East exit
.Use Stairs
PPurchase Items From Shop
DDrop Item/Equipment
  • Use LEFT/RIGHT arrows to select player who will drop item
  • Use UP/DOWN arrows to select item to drop
  • Press D key to drop selected item
  • Press ESC to exit drop screen
GGet Items On Floor
  • Party items like Gold and Rations are automatically added when "got"
  • Individual items must be given to specific players
  • Select the item to get with the UP/DOWN arrows
  • Give item to a player by pressing the number key that corresponds to the number below them
  • Players cannot be given items they cannot equip, or which would exceed the limit of their inventory size
TTrade Between Players
QBalance Quivers (distributes ammo between players with appropriate weapons)
COpen Chest
VOpen Vault
  • Select player who will open vault from list (except in solo mode)
  • Enter guess at vault code using number keys
  • Give up on vault with ESC key
FDrink From Fountain
AAsk Living Statue About An Item
NNegotiate
SSearch Room For Hidden Doors
Combat Keys
ARROWSMove active player into empty space, or attack adjacent enemy with melee weapon
UUse Item
  • If item affects one player, select which player from list that is shown (solo mode games automatically pick only player)
  • If item effects one enemy, game switches to Fire mode
FFire Weapon or Item
  • Use ARROWS to move targetting reticle onto desired enemy
  • Use F key to fire when ready
SPCEnd Current Player Turn (in combat)
ESCCancel Current Action
Shop Keys
LEFT/RIGHTChange Current Buying Player
1Shop For Weapons
2Shop For Ranged Weapons
3Shop For Body Armor
4Shop For Special Armor
5Shop For Healing
6Shop For Rations
ESCExit Current Menu/Exit Shop (when at top menu)
